✅ MASTER PROMPT FOR INDIVIDUAL ASSIGNMENT
(Fully meets SITER Internship Requirements)

Role: You are a frontend web developer.
Task: Build a Fully Responsive Static Website for an Education Feedback & Review Platform as part of an internship individual assignment.

Project Objective

Design and develop a static website that allows students/users to submit feedback and reviews for educational institutions or courses, view ratings, and demonstrate admin-level feedback analytics (UI only). The website should focus on usability, responsiveness, SEO, and clean design.

Mandatory Requirements
1. Design (Figma)

Create a complete wireframe using Figma before coding.

Include wireframes for:

Home page

About page

Services / Reviews page

Contact page

Feedback form page

Export the wireframe as PDF and upload it to GitHub.

2. Pages & Sections (HTML + CSS)

The website must include the following:

Header with navigation bar

Sticky navigation bar using CSS

Footer

Home page

About page

Services / Business page (Education reviews)

Contact page

Interaction page (Feedback form)

Navigation:

Use ID-based navigation for smooth scrolling.

Ensure all pages are properly linked.

3. Features (Static Implementation Allowed)

Feedback submission form (UI only)

Anonymous or logged-in feedback option (checkbox based)

Rating system (1–5 stars using HTML/CSS)

Admin dashboard page (static mockup)

Feedback analytics section (charts / cards with dummy data)

4. Styling & UI

Use HTML, CSS, Bootstrap and/or Tailwind CSS

Choose 2 color themes:

Primary color

Secondary color

Use Google Fonts:

One font for headings

One font for body text

Use images from Unsplash

Use dummy text where required

5. Responsiveness

Follow mobile-first approach

Website must work on:

Mobile

Tablet

Desktop

Use responsive layouts and media queries

6. SEO Optimization

Use semantic HTML tags (header, nav, section, footer, etc.)

Add meta title & meta description

Use proper heading hierarchy (H1 → H6)

Optimize images with alt attributes

7. Add-Ons (Any One)

Embed a YouTube video

OR embed Google Maps

OR add any creative interactive section

8. Version Control & Hosting

Use Git for version control

Push code to GitHub or GitLab

Host website using:

GitHub Pages / Netlify / Vercel
