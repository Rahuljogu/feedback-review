# Education Feedback & Review Platform

This project is a **static, responsive website** developed as part of the **SITER Internship Individual Assignment**.
The website is designed to collect student feedback and reviews for educational institutions or courses and display them in a clean and user-friendly interface.

The project focuses on frontend development skills, responsive design, and basic UI-based analytics without using any backend or database.

---

## Project Purpose

The main goal of this project is to:

* Allow students to share feedback and ratings
* Display education reviews in a structured way
* Showcase a static admin dashboard for feedback analysis
* Follow proper HTML, CSS, and SEO practices

---

## Technologies Used

* HTML5
* CSS3
* Bootstrap / Tailwind CSS
* Google Fonts
* Unsplash Images
* Figma (for wireframe design)

---

## Website Pages

* **Home Page** – Overview of the platform
* **About Page** – Information about the website
* **Services / Reviews Page** – Education reviews and ratings
* **Feedback Page** – Feedback submission form
* **Contact Page** – Contact details and form
* **Admin Dashboard Page** – Static feedback analytics UI

---

## Features

### Feedback Form (UI Only)

* Optional name field
* Institution or course name input
* Star rating system (1–5 stars)
* Anonymous feedback option
* Feedback text area

### Rating System

* Implemented using HTML and CSS
* Interactive star-based ratings

### Admin Dashboard (Static)

* Total feedback count display
* Average rating section
* Analytics cards with dummy data
* Recent feedback list

---

## Responsiveness

* Mobile-first design approach
* Works smoothly on:

  * Mobile devices
  * Tablets
  * Desktop screens

Responsive layouts are created using flexible grids and media queries.

---

## SEO & Accessibility

* Semantic HTML elements (`header`, `nav`, `section`, `footer`)
* Proper heading structure (H1 to H6)
* Meta title and description added
* Images include `alt` attributes
